# Pixel Hero Adventure 游戏脚本

本项目为 **Pixel Hero** 的自动化游戏脚本，专为 **BlueStacks 模拟器** 环境设计，旨在简化玩家日常操作，提高游戏效率。

## 模拟器设置

1. 下载 BlueStacks 模拟器：[官方链接](https://www.bluestacks.com)
2. **模拟器窗口命名**：
   - 改名为 `001`, `002` 等序号，便于脚本管理。
3. **模拟器多开设置**：
   - **分辨率调整**：
     - 2560x1440 屏幕：一排排列 4 个窗口。
     - 1920x1080 屏幕：一排排列 3 个窗口。
   - **硬件配置**：
     - 内存和核心数选择 **中等**。
     - DPI 设置为 **160**。
     - 模拟器分辨率选择 **竖直模式**（1080x1920）。

---

## 环境配置

1. 安装项目所需依赖：
   pip install -r requirements.txt
  
2. 将根目录下的 `env_example` 文件重命名为 `.env`。.env中的参数为循环时间和窗口、地图信息，比如001=35表示第一个窗口刷第3个地图第5个关卡

---

## 项目结构说明

- **`dm.dll` 和 `Regdm.dll`**：
  - 大漠插件及其注册文件，提供模拟器操作支持。
  - 在电脑查杀那里要设置这两个文件所在的文件夹为免杀区,要不然会被隔离清除
- **OCR 文件夹**：
  - 包含文字识别软件，运行脚本前需启动 `OCR.exe` 文件以建立本地服务器，随时监听脚本中的截图文字识别请求。
- **主要代码文件**：
  - `libs/game.py`：核心游戏逻辑。
  - `libs/schedule.py`：任务调度与自动化管理。
- **测试文件夹**：
  - `test/` 中为临时测试脚本，不用于实际运行。

## 脚本功能逻辑

### 日常循环操作
- 每隔数分钟自动执行：
  - 领取钻石。
  - 拆解装备获取经验，并将装备加入图鉴以提升战斗力。
  - 放置 **Toyz** 侦查以获取奖励。

### 定时任务
- 每 12 或 24 小时自动运行：
  - 领取免费钻石。
  - 领取邮件奖励。
  - 领取任务奖励。
  -罗马竞技场和清理地下城
  -后期理论上还可以结合A星寻路刷野外boss

---

## 注意事项

1. BlueStacks窗口的高度尽量保证有1080p，以确保视图准确。
2. 启动脚本前，务必先运行 `OCR.exe`，否则文字识别功能将无法工作。
